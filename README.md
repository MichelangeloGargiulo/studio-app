## 📋 Cosa fa
App desktop per elaborare i dati raccolti con i form mobile e generare report professionali.

- 📊 **Panoramica** con KPI e grafici
- 👤 **Analisi persone** chiave e sogni
- 🏦 **Patrimonio** attuale e proiezioni
- ⚠️ **Mappa dei rischi** con soluzioni assicurative
- 📈 **Scenari futuri** (base, ottimistico, pessimistico)
- 🎯 **Piano d'azione** con priorità
- 📋 **Dati raccolti** completi

## 📥 Importa dati
- Incolla il codice dall'app mobile
- Carica file `.json`
- Usa i dati **Demo** per test

## 📤 Esporta
- **Excel** con 3 fogli (Riepilogo, Proiezioni, Dati)
- **Stampa PDF** del report completo
- **WhatsApp** per condividere la sintesi

## 🏠 Famiglie
- Curva di Modigliani (ciclo di vita)
- Gap protezione vita, malattia, invalidità, LTC
- Proiezione patrimonio fino alla pensione

## 🏢 Aziende
- Corporate Life Cycle (Damodaran)
- Enterprise Value e DCF
- Key person risk, successione, separazione patrimoni

## 🛠️ Tecnologia
- **Chart.js** per i grafici
- **SheetJS** per l'export Excel
- **100% locale** – nessun dato lascia il computer
- **Nessun database** richiesto
